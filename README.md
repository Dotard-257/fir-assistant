# FIR Filing Assistant 🇮🇳

An AI-powered legal aid tool that helps Indian citizens draft FIRs 
in correct legal language based on the Bharatiya Nyaya Sanhita (BNS) 2023.

## Features
- AI chatbot that drafts FIRs in formal legal language
- Identifies correct BNS / IPC sections automatically
- 12 Latin legal terms with explanations
- 9 citizen rights including Zero FIR and arrest rights
- BNS 2023 quick reference across 4 crime categories
- Works in English and Hindi
- Copy and Print buttons for FIR draft

## Tech Stack
- Backend: Python, FastAPI, Uvicorn
- Frontend: HTML, CSS, JavaScript
- AI: Groq API (LLaMA 3.3 70B)
- Legal Framework: BNS 2023, BNSS 2023, BSA 2023

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the server:
   python main.py

3. Open browser at:
   http://localhost:8000

## Social Impact
Helps citizens who don't know legal language to file FIRs correctly,
reducing dependence on expensive lawyers for basic legal procedures.
Especially useful for rural and underserved communities in India.

## Legal Disclaimer
This is a legal guidance tool, not a substitute for a lawyer.
